# Redrob NEXUS Intelligent Candidate Ranker

> **Hackathon Entry** — Intelligent Candidate Discovery & Ranking Challenge

**NEXUS** (Neural + Expert Cross-signal Unified System) goes far beyond keyword matching. It understands the *arc* of a candidate's career, the *depth* of their skill cluster coverage, and the *behavioral signals* that indicate genuine availability — all within 60 seconds on a CPU.

---

## Architecture

```
candidates.jsonl (100K)
        │
        ▼
┌─────────────────────────────────────────────────────────┐
│  OFFLINE (nexus_precompute.py — run once, no time limit)│
│                                                         │
│  SentenceTransformer (all-MiniLM-L6-v2)                 │
│    → candidate_embeddings.npy  (N × 384 float32)        │
│    → jd_embedding.npy          (1 × 384 float32)        │
│  TF-IDF Vectorizer             → tfidf_matrix.npz       │
│  JD Analysis                   → jd_analysis.json       │
└─────────────────────────────────────────────────────────┘
        │ artifacts/nexus/
        ▼
┌─────────────────────────────────────────────────────────┐
│  RUNTIME (nexus_ranker.py — ≤5 min, CPU only, no net)  │
│                                                         │
│  Layer 1: Career Arc Intelligence ← time-decayed traj. │
│  Layer 2: Skill Cluster Coverage ← 5 JD-derived groups │
│  Layer 3: Semantic Similarity   ← embedding dot product │
│  Layer 4: Experience Intelligence ← YoE + edu + domain │
│  Layer 5: Location + Logistics  ← geo + notice + mode  │
│  Behavioral Multiplier          ← all 23 signals        │
│  Graded Anti-Pattern Detection  ← suspicion score 0→1  │
│                                                         │
│  → submission.csv (100 ranked candidates)               │
└─────────────────────────────────────────────────────────┘
```

## Scoring Components

| Component | Weight | Innovation |
|---|---|---|
| **Career Arc** | 30% | Time-decayed trajectory — recent ML roles at product companies weighted 3× more than old ones |
| **Skill Clusters** | 25% | 5 semantic clusters (Vector/Retrieval, LLM/RAG, Training, Production, Data) — depth rewarded |
| **Semantic** | 25% | Sentence-transformer cosine similarity (all-MiniLM-L6-v2); falls back to TF-IDF |
| **Experience** | 12% | YoE band (5–9 ideal), applied ML ratio, education tier bonus |
| **Location** | 8% | City fit, notice period, relocation, work mode |
| **Behavioral ×** | mult | All 23 redrob_signals with time-decay; range 0.25–1.30 |
| **Anti-Pattern** | factor | Graded suspicion score 0→1 — graceful degradation, not binary cap |

### Key Innovations

- **Career Arc Intelligence**: Exponential time decay (`exp(-0.15 × years_ago)`) — a candidate who pivoted to AI/ML 6 months ago ranks higher than someone who peaked 5 years ago
- **Skill Cluster Coverage**: 5 semantic clusters replace flat keyword lists. Log-scaled depth scoring rewards genuine expertise
- **Semantic Embeddings**: `all-MiniLM-L6-v2` (384-dim, L2-normalized) for true semantic understanding; TF-IDF bigrams as automatic fallback
- **Graded Anti-Patterns**: Each detected inconsistency adds suspicion weight (0→0.95) — fair to candidates with 1 inconsistency
- **Founding Team Fit**: Startup experience (company_size 1–50) explicitly rewarded in career arc
- **Word-boundary skill matching**: Prevents false positives (e.g. `"ai"` inside `"faiss"`)
- **Score clamped to [0, 1]**: `final = min(1.0, base × bm × anti_factor)`

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Precompute (once — ~5-8 min for 100K with sentence-transformers, ~2 min TF-IDF only)
python nexus_precompute.py --candidates ./candidates.jsonl

# 3. Rank (≤60s for 100K on CPU)
python nexus_ranker.py --candidates ./candidates.jsonl --out ./submission.csv

# 4. Validate
python India_runs_data_and_ai_challenge/validate_submission.py ./submission.csv

# 5. Sandbox UI
streamlit run nexus_app.py
```

## Compute Profile

- **Precomputation (sentence-transformers)**: ~5–8 min / 100K on CPU (one-time)
- **Precomputation (TF-IDF fallback)**: ~2 min / 100K on CPU
- **Runtime ranking**: ~15–45 sec / 100K on CPU
- **RAM**: ~2–4 GB
- **No GPU, no network required at ranking time**
